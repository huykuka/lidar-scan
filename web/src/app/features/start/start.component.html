Key Features

Node-Based Processing

Create complex processing pipelines with modular nodes: sensors, fusion, filtering, segmentation, and more.

Real-Time 3D Visualization

High-performance WebGL rendering with Three.js. Handle dense point clouds at 60+ FPS.

Multi-Sensor Fusion

Combine data from multiple LiDAR sensors with automatic synchronization and transformation.

Advanced Recording

Record full N-dimensional point cloud data in compressed PCD format. Preserve intensity, reflector, and all fields.

Pipeline Operations

Crop, downsample, filter outliers, segment planes, cluster objects, detect boundaries, and more.

WebSocket Streaming

Low-latency binary protocol for efficient point cloud transmission over WebSockets.
