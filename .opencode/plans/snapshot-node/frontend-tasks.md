# Snapshot Node — Frontend Tasks

> No frontend work required for this feature.
> 
> The Snapshot Node has no Angular component, settings panel, or UI trigger.
> All interaction is via the HTTP API (`POST /api/v1/nodes/{node_id}/trigger`).
> 
> See `requirements.md` § "Out of Scope" for details.
> Frontend UI is explicitly deferred to a future iteration.
